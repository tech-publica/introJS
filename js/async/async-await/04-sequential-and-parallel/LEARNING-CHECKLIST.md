# Learning Checklist

- [ ] I can distinguish dependent operations from independent operations.
- [ ] I can explain why consecutive `await` expressions run these tasks sequentially.
- [ ] I can start multiple independent operations before awaiting their results.
- [ ] I can combine independent promises with `Promise.all`.
- [ ] I can destructure the result array returned by `Promise.all`.
- [ ] I understand that `await` does not automatically make operations parallel.
- [ ] I can choose the appropriate structure for a new example.

Run both demonstrations and explain the difference in elapsed time. Exact times
may vary slightly, but the reason for the difference should be clear.
