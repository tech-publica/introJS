# Learning Checklist

- [ ] I can explain what happens when an awaited promise rejects.
- [ ] I can use `try` around asynchronous operations that may fail.
- [ ] I can use `catch` to receive and display an error.
- [ ] I can use `finally` for cleanup that must happen after either outcome.
- [ ] I know that code after the failed `await` inside `try` is skipped.
- [ ] I understand why rejecting with an `Error` object is helpful.
- [ ] I can explain why the button is re-enabled in `finally`.

Test both available outcomes. Being able to explain only the successful path is
not enough for this lesson.
