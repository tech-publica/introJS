# Learning Checklist

- [ ] I can rewrite a `.then()` chain using `async` and `await`.
- [ ] I can explain why the second request must wait for the first request.
- [ ] I can identify the value produced by each `await` expression.
- [ ] I remember that the async function still returns a promise.
- [ ] I can explain why two consecutive `await` expressions are correct here.
- [ ] I can follow the character and planet values through the function.

Do not move on until you can point to the exact line that creates each request
and the exact line that receives each response value.
