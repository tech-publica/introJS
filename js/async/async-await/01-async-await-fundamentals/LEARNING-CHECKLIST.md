# Learning Checklist

Before finishing this lesson, check whether you can honestly say:

- [ ] I can identify an async function by the `async` keyword.
- [ ] I can explain why calling an async function always returns a promise.
- [ ] I can explain how `return value` determines that promise's fulfillment value.
- [ ] I can use `await` to obtain a fulfilled promise's value.
- [ ] I can await both `fetch()` and `response.json()`.
- [ ] I understand that a Fetch `Response` is not yet the parsed body.
- [ ] I can check `response.ok` before using a response.
- [ ] I understand that `await` pauses only the current async function.
- [ ] I understand that other JavaScript can continue while that function waits.
- [ ] I can predict the main execution order in the demonstration.

If an item is unclear, rerun the demonstration, inspect the console, and explain
the relevant line to another student before moving to lesson 2.
