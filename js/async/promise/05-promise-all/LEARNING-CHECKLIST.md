# Learning Checklist

- [ ] I can distinguish independent operations from dependent operations.
- [ ] I can start all independent requests without waiting between them.
- [ ] I can combine an array of promises with `Promise.all`.
- [ ] I understand when the combined promise fulfills.
- [ ] I can explain why result values remain in input order.
- [ ] I can explain the fail-fast rejection behavior of `Promise.all`.
- [ ] I understand that rejection does not automatically cancel other requests.
- [ ] I can decide whether a problem needs a chain or `Promise.all`.

You have met the lesson goals when you can justify the structure from the data
dependencies, rather than choosing it only from the number of requests.
