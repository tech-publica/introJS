# Learning Checklist

- [ ] I can explain why asynchronous `xhr.send()` returns before the response arrives.
- [ ] I can identify `handleLoad` as a callback function.
- [ ] I can explain that the browser invokes the callback when the response arrives.
- [ ] I can predict the three numbered console messages in the correct order.
- [ ] I understand why the clock and counter remain responsive during the request.
- [ ] I can distinguish a successful HTTP response from a network error.
- [ ] I can explain why code outside the callback cannot immediately use the response.

You are ready to continue when you can identify who calls each function and
which work happens now versus later.
