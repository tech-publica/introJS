# Learning Checklist

Use this checklist before the lesson to preview its goals and after the lesson
to check your understanding.

- [ ] I can explain what AJAX allows a page to do.
- [ ] I can identify the request URL and HTTP method in the XHR code.
- [ ] I can explain what the third argument of `xhr.open(..., false)` means.
- [ ] I can explain why `xhr.send()` blocks in this example.
- [ ] I can describe what happens to the page clock and controls while blocked.
- [ ] I can read the HTTP status and parse the JSON response.
- [ ] I understand that synchronous XHR is shown only for comparison and should
      not be used on the browser's main thread.

You are ready to continue when you can explain why the response is available on
the next line, and what the browser sacrifices to make that possible.
