# Learning Checklist

- [ ] I can declare and initialize a private field using a `#` name.
- [ ] I can explain why `_name` is a convention rather than real privacy.
- [ ] I understand that `name` and `#name` are different fields.
- [ ] I can expose private state through a getter.
- [ ] I can validate a proposed value in a setter before storing it.
- [ ] I can use accessor property syntax from outside the class.
- [ ] I can create a read-only public interface with a getter and no setter.
- [ ] I can compare JavaScript private fields and accessors with Java equivalents.

You are ready to continue when you can protect an invariant—such as a minimum
temperature—without allowing callers to change the private field directly.
