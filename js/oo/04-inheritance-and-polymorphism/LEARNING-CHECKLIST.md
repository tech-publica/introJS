# Learning Checklist

- [ ] I can create a subclass with `extends`.
- [ ] I can call the parent constructor with `super(...)`.
- [ ] I understand why `super(...)` must run before using `this` in a derived constructor.
- [ ] I can override an inherited method with a method of the same name.
- [ ] I can call the parent implementation with `super.method()`.
- [ ] I can explain the difference between `super.method()` and `this.method()`.
- [ ] I can recognize polymorphism when different objects receive the same method call.
- [ ] I can explain how runtime prototype lookup selects an implementation.
- [ ] I can compare JavaScript inheritance with Java's class-based type system.

You have met the goals when you can build a small hierarchy and explain which
method implementation runs for every object in a mixed collection.
