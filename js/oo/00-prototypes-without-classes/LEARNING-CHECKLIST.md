# Learning Checklist

- [ ] I can explain prototype delegation in my own words.
- [ ] I can draw the lookup chain from a dog object to `Object.prototype`.
- [ ] I understand that inherited methods are looked up rather than copied.
- [ ] I can use `Object.create` to connect one object to another prototype.
- [ ] I can distinguish instance data from shared prototype behavior.
- [ ] I can explain the role of constructor functions and `new`.
- [ ] I can explain how factory functions create objects without `new`.
- [ ] I can connect prototypes to `class`, `extends`, and `super` syntax.

Complete the vintage and modern checklists as part of this lesson. You are ready
to move on when you can compare their object-creation styles while recognizing
that both rely on prototype delegation.
