# Learning Checklist

- [ ] I can explain the difference between a class and an object instance.
- [ ] I can declare a class using the `class` keyword.
- [ ] I can use a constructor to initialize each instance's properties.
- [ ] I can explain what `this` refers to inside an instance method.
- [ ] I can create several independent instances with `new`.
- [ ] I can declare and call an instance method.
- [ ] I can use `instanceof` to check how an object was created.
- [ ] I can explain why changing one instance does not change another instance.

You are ready for the next lesson when you can create a small class from scratch
without copying the `Student` example.
