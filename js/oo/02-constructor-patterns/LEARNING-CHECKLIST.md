# Learning Checklist

- [ ] I can explain why JavaScript classes cannot declare overloaded constructors.
- [ ] I can use a default parameter for a small number of optional values.
- [ ] I can use an options object for several optional or named values.
- [ ] I can supply defaults while destructuring an options object.
- [ ] I can define and call a static factory method.
- [ ] I understand that a static method is called on the class, not an instance.
- [ ] I can choose among defaults, an options object, and named factory methods.
- [ ] I can compare these patterns with constructor overloading in Java.

You have met the lesson goals when you can justify which construction pattern is
clearest for a new set of required and optional values.
